"Who Are You" is a simple and interesting web app which helps you guess who you are from an image. It runs in PHP and employs the powerful face detection API from AlchemyAPI service on Bluemix. With only a few code lines, you are able to build up a similar app with fantastic machine learning features. Source code could be found at https://hub.jazz.net/project/chunbintang/whoru/overview.
